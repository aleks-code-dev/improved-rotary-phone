   - Import `jsonata from 'jsonata'` (1.8.7 — synchronous evaluate, NOT 2.x async)
   - Import types from `../../shared/schemas/collection.js` and `../ipc/channels.js`
   - Define `REF_REGEX = /\{\{step(\d+)\.response\.(body|headers|status)(.*?)\}\}/g`
   - Export `resolveReferences(spec: RequestSpec, priorResults: StepRunResult[]): { request: RequestSpec; warnings: Array<{ reference: string; reason: string }> }`
   - Implementation:
     - `resolveString(template: string)`: replaces all `{{stepN.response.X.path}}` matches
     - For each match: find priorResult by stepIndex (1-based), extract body/headers/status
     - Body: decode base64 → JSON.parse → evaluate JSONata path
     - Headers: convert array to object → evaluate JSONata path
     - Status: return as string directly
     - If path empty: return whole body as string
     - If step not run or path not found: push warning, return empty string (per D-14)
     - If JSONata throws: push warning with error message, return empty string
   - Apply resolveString to: spec.url, each header value, spec.body.text (if mode='raw')
   - Return resolved request + warnings array

| T-04-03 | Denial of Service | Infinite chain loop / circular refs | mitigate | Pre-run validation detects circular references; DFS with WHITE/GRAY/BLACK states halts before execution |

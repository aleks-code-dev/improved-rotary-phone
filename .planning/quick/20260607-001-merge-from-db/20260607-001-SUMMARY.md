# Quick Task 20260607-001: Merge from DB button with camelCase matching and Monaco line-highlight fade-out

**Result:** Implemented a "Merge from DB" button in BodyTab that matches DB row values to DTO JSON fields via camelCase conversion and highlights matched lines in the Monaco editor with a 3-second fade-out animation.

## Files Changed

### New
- `src/renderer/lib/textCase.ts` — `toCamelCase()` and `buildReverseCamelCaseMapping()` utilities
- `src/renderer/components/RequestEditor/BodyTab.css` — CSS keyframe animation for fade-out highlight

### Modified
- `src/renderer/components/RequestEditor/BodyTab.tsx` — Added Merge button, Monaco editor ref for decorations, formatJson helper, highlight/fade-out logic

## Implementation Details

1. **`toCamelCase`** converts snake_case/kebab-case to camelCase (e.g., `user_name` → `userName`)
2. **`buildReverseCamelCaseMapping`** creates a reverse lookup from camelCase DB key → original DB key
3. **Merge logic** in `handleMergeFromDb`: 
   - Takes current DTO body text, parses as JSON
   - For each DTO field, converts DB keys to camelCase and checks for a match
   - Assigns DB value to matched DTO field
   - Uses `JSON.stringify(merged, null, 2)` for formatted output
4. **Monaco decorations**: After merge, calls `highlightMatchedLines()` which uses `editor.deltaDecorations()` with CSS class `monaco-matched-field-highlight`
5. **Fade-out**: CSS `@keyframes highlightFadeOut` animates background-color from green to transparent over 3 seconds
6. **All 3 buttons format output**: DTO generation, DB row generation, and Merge all use `formatJson()` (parse + stringify with 2-space indent)

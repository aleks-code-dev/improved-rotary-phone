# Quick Task 20260607-001: Add "Merge from DB" Button with CamelCase Matching

**Description:** Add a third "Merge from DB" button that takes the DTO JSON structure and DB row JSON, converts DB field names to camelCase, matches them with DTO JSON fields, and assigns DB values. Show visual indicator for matched fields.

## Tasks

### 1. Create camelCase conversion utility (`src/renderer/lib/textCase.ts`)
- Implement `toCamelCase(str: string): string` - converts snake_case, PascalCase, etc. to camelCase
- Implement helper for object key transformation

### 2. Add "Merge from DB" button to `BodyTab.tsx`
- Add third button between "Generate from DTO" and "Generate from DB row"
- Button label: "Merge from DB" 
- Disabled when: no DTO detected, no DB row selected, or helper offline
- On click: call merge logic

### 3. Implement merge logic in `BodyTab.tsx`
- Get the current DTO JSON (already in body text)
- Get the DB row JSON (from selectedRow)
- Convert DB field names (snake_case) to camelCase
- For each field in DTO JSON:
  - Check if a camelCase-converted DB field matches
  - If yes, assign the DB value to the DTO field
- Track which fields were matched (for visual indicator)

### 4. Add visual indicator for matched fields
- After merge, show a small toast/indicator showing how many fields were matched
- Show the matched field names in a tooltip or small label
- Use green/success color for the indicator

### 5. Update tests (if any exist for BodyTab)
- Verify the new button renders correctly
- Verify merge logic handles edge cases (null values, nested objects, etc.)

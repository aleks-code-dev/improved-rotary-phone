Modify `handleSave` (around line 173) to:

1. Look up the active tab from `useTabs.getState().openTabs`
2. If the tab has BOTH `sourceCollectionId` and `sourceItemIndex` defined:
   a. Serialize the current `spec` to Postman v2.1 item format (same logic as SaveAsModal lines 24-44)
   b. Read the full collection via `window.api.collections.read({ id: tab.sourceCollectionId })`
   c. Replace `coll.item[tab.sourceItemIndex]` with the serialized item
   d. Write back via `window.api.collections.update({ id: tab.sourceCollectionId, collection: coll })`
   e. Call `useTabs.getState().markClean(tabId)` to remove the orange dot
   f. Invalidate queries so sidebar refreshes: `queryClient.invalidateQueries({ queryKey: ['collections'] })` and `queryClient.invalidateQueries({ queryKey: ['collections', tab.sourceCollectionId] })`
   g. Update tab name to match current method+url via `useTabs.getState().updateTab(tabId, { sourceItemName: spec.method + ' ' + spec.url })`
3. If the tab has NO source info (new unsaved tab), open `setSaveModalOpen(true)` as before.

**Step B — Serialization helper for spec→collection-item:**
Extract the serialization logic from SaveAsModal into a standalone function that both handleSave and SaveAsModal reuse. Place it in a new file or inline. The function signature:

```typescript
function specToCollectionItem(spec: RequestSpec, name: string): any {
  return {
    name: name || 'Untitled Request',
    request: {
      method: spec.method,
      url: spec.url || '',
      header: (spec.headers || []).map(h => ({ key: h.key, value: h.value })),
      body: spec.body?.mode !== 'none' ? {
        mode: spec.body?.mode,
        raw: spec.body?.mode === 'raw' ? (spec.body as RequestBodyRaw).text : undefined,
        urlencoded: spec.body?.mode === 'urlencoded' ? (spec.body as RequestBodyUrlencoded).fields : undefined,
        formdata: spec.body?.mode === 'form-data' ? (spec.body as RequestBodyFormData).fields : undefined,
      } : undefined,
      auth: spec.auth?.type !== 'none' ? spec.auth : undefined,
    },
    response: [],
    event: [],
    ...(spec.queryParams?.length > 0 ? { _queryParams: spec.queryParams } : {}),
  };
}
```

Put this helper in `src/renderer/components/RequestEditor/specToItem.ts` and import it in both `RequestEditor.tsx` and `SaveAsModal.tsx`. Replace the inline serialization in SaveAsModal.handleSave with a call to this helper.
